# Shahreyar - Senior WordPress Engineer

Portfolio and personal website built with React, Vite, and Tailwind CSS.

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the local development server:
```bash
npm run dev
```

### Build for Production

Generate a production build:
```bash
npm run build
```

Generate a production zip file:
```bash
npm run build:zip
```

### Deployment to GitHub Pages

To deploy the site to GitHub Pages:
```bash
npm run deploy
```
